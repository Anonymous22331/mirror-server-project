Реализация сервиса сетевых сообщений поверх Mirror в Unity.
Добавлена система подписок клиентов на типы сообщений с серверной фильтрацией и подтверждением.
Реализация DI с помощью Zenject.

**Для проверки работы**
1. Скачать Release билд.
2. Склонировать репозиторий из ветки develop
3. Запускаем проект в Editor, выбираем Server Only в окне
4. Запускаем билд из релиза, выбираем Client
5. Все сообщения можно увидеть в консоли в Editor
